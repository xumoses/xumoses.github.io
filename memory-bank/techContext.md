# Tech Context

## Technical Stack

- **Language**: TypeScript 6.0 Beta (ES Modules).
- **Framework**: React 19.
- **Build Tool**: Vite 8 (Stable).
- **Styling**: Tailwind CSS 4.
- **Routing**: React Router 7.
- **Linter**: oxlint (with React plugin).
- **Formatter**: oxfmt.

## Development Environment

- **Package Manager**: pnpm.
- **Runtime**: Node.js.
- **OS**: macOS (specifically configured for macOS Tahoe environment).

## Dependencies

- `clsx`: Utility for constructing className strings.
- `react-dom`: Entry point to the DOM and server renderers for React.
- `gh-pages`: For easy deployment to GitHub.
- `simple-git-hooks`: For Git hooks (linting/formatting before commit).

## Architecture

- Client-side Single Page Application (SPA).
- Static assets served from `public/` and `src/assets/`.
