# Product Context

## Why this project exists?
This project is a personal website and portfolio (mo-space) hosted on GitHub Pages (xumoses.github.io). It serves as a hub for showcasing projects, sharing knowledge, and potentially hosting small games or interactive experiences.

## Problems it solves
- Provides a centralized location for professional and personal online presence.
- Demonstrates technical skills through a live, interactive web application.
- Offers a platform to share knowledge and experiments.

## How it should work
- The site is built with React and Vite, utilizing Tailwind CSS for styling.
- It features a multi-page navigation system (Home, Game, Knowledge).
- It is designed to be responsive and performant.
- Deployment is automated via GitHub Pages.
